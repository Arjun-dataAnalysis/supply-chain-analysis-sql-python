# 📊 Supply Chain Analysis using SQL & Python

## 📌 Project Overview

This project focuses on analyzing supply chain operations to identify key business insights related to **revenue performance, transportation costs, inventory management, supplier efficiency, and delivery operations**.

The analysis was performed using **SQL for data exploration and KPI analysis** and **Python for data cleaning, exploratory data analysis (EDA), visualization, and insight generation**.

The objective of this project is to understand supply chain patterns and provide data-driven recommendations to improve operational efficiency and reduce unnecessary costs.

# 💡 Key Business Insights

### 🚚 Transportation Optimization

* Different transportation routes showed variations in average transportation costs.
* Identifying expensive routes can help organizations optimize logistics planning.

### 📦 Inventory Management

* Stock levels vary across products, highlighting opportunities for better inventory planning.
* Maintaining optimal inventory can reduce storage costs and prevent shortages.

### 💰 Cost Management

* Transportation and manufacturing costs contribute significantly to overall supply chain expenses.
* Cost analysis helps identify areas where operational improvements can reduce expenses.

### ⏱️ Delivery Performance

* Shipping time and lead time analysis helps evaluate delivery efficiency.
* Reducing unnecessary delays can improve customer satisfaction.
